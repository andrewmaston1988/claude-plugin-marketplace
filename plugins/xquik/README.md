# xquik

Xquik guidance for X data extraction, automation, webhooks, REST API, and MCP workflows.

## Included skill

| Skill | Use it for |
|---|---|
| xquik | Planning Xquik API, MCP, webhook, monitoring, and data extraction workflows |

## Install

After registering this marketplace, install the plugin:

```bash
claude plugin install xquik@andrewmaston1988-claude-plugins
```

Or load it from a local checkout:

```bash
claude --plugin-dir /path/to/claude-plugin-marketplace/plugins/xquik
```

## Configuration

Configure an Xquik API key in your project environment before asking Claude to make live API requests. Keep live requests explicit and use the public docs for current endpoint and MCP setup details.
